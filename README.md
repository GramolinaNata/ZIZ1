# ZIZ1

Задача: вывывести апишку с помощью react query и сделать в ней поиск

## Стек

React 18, TypeScript, Vite, TanStack React Query, Axios, MockAPI

## Запуск
```bash
npm install
npm run dev
```

## Функциональность

- Получение списка пользователей через React Query
- Поиск пользователей по имени в реальном времени
- API подключено через MockAPI
